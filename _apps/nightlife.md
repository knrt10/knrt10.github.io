# How to use

- clone the repo

- run `npm install`

- add `.env file`

Your env file should have

```
GITHUB_KEY=GITHUB_KEY
GITHUB_SECRET=GITHUB_SECRET
MONGO_URI=Your mongodb url
APP_URL=http://localhost:8080/
```

- run `node server.js`
